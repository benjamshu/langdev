#  斤丸舌 :: PHONOLOGY  #

For simplicity, a Latin orthography is used to record FIZENG phonologies, the IPA equivalencies for which are provided below:

###  Vowels:  ###

| Latin |  IPA  |
| ----: | :---- |
|   u   |  /u/  |
|   e   |  /e/  |
|   i   |  /i/  |
|   a   |  /a/  |
|   o   |  /ɒ/  |

###  Consonants:  ###

| Latin |  IPA  |
| ----: | :---- |
|   p   |  /p/  |
|   b   |  /b/  |
|   m   |  /m/  |
|   h   |  /h/  |
|   f   |  /ɸ/  |
|   r   |  /ɾ/   |
|   d   |  /d/  |
|   n   |  /n/  |
|   q   |  /ŋ/  |
|   s   |  /ɕ/  |
|   z   |  /ʑ/  |
|   k   |  /k/  |
|   w   |  /w/  |
|   y   |  /j/  |

Note that f~h, d~r, n~q, and z~s are not considered distinct sounds in FIZENG.
